設定 Google Analytics

要使用數據分析功能，你需要先設定 Google Analytics 4。以下是設定步驟：

  1. 前往 Google Analytics 建立帳戶
  2. 建立新的 GA4 資源（屬性），網站填入 https://tediousman.github.io
  3. 取得「測量 ID」（格式為 G-XXXXXXXXXX）
  4. 在下方輸入你的測量 ID 並儲存

儲存後需要將測量 ID 更新到 _config.yml 並重新部署網站。
GA 通常需要 24-48 小時才會開始顯示數據。

連接 Google Analytics API 以查看詳細數據（國家、停留時間等）

Client ID 格式：xxxxxxxxx-xxxx.apps.googleusercontent.com
